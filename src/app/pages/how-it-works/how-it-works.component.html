❓ Questions fréquentes

Tout ce que vous devez savoir sur EventPhoto

Comment accéder aux photos d'un événement ?

Chaque événement dispose d'un code unique. Il suffit d'entrer ce code sur notre plateforme ou de scanner le QR code fourni par l'organisateur pour accéder à toutes les photos de l'événement.

Comment sont partagés les revenus ?

Lors de la création d'un événement, vous définissez le pourcentage de chaque bénéficiaire. Les revenus sont automatiquement répartis selon ces pourcentages à chaque vente.

Les photos sont-elles sécurisées ?

Absolument ! Toutes les photos sont stockées de manière sécurisée et seules les personnes disposant du code d'événement peuvent y accéder. Chaque événement est protégé par son code unique.

Combien coûte EventPhoto ?

EventPhoto prend une commission sur chaque vente réalisée. Pas de coûts fixes, vous ne payez que quand vous vendez !
